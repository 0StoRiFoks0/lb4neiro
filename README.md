# Лабораторна робота №4
## Кластеризація даних за допомогою нейронних мереж Кохонена

### Мета роботи
Дослідити принцип роботи нейронних мереж Кохонена, що самоорганізуються (Self-Organizing Maps, SOM), та виконати кластеризацію двовимірних і тривимірних даних у середовищі MATLAB.

Використані технології

- MATLAB
- Deep Learning Toolbox
- Self-Organizing Maps (SOM)
- Competitive Learning

#Завдання лабораторної роботи

1. Згенерувати двовимірні кластеризовані дані.
2. Створити шар Кохонена.
3. Виконати навчання мережі.
4. Перевірити кластеризацію тестових даних.
5. Побудувати карту Кохонена.
6. Згенерувати тривимірні кластери.
7. Побудувати SOM для 3D даних.

Основні функції MATLAB

| Функція | Призначення |
|---|---|
| `competlayer()` | створення шару Кохонена |
| `selforgmap()` | створення карти Кохонена |
| `train()` | навчання мережі |
| `sim()` | моделювання роботи мережі |
| `vec2ind()` | перетворення виходу у номер кластера |
| `plotsomtop()` | відображення топології |
| `plotsomnd()` | відображення відстаней |
| `plotsomplanes()` | площини ваг |
| `plotsompos()` | розташування нейронів |
